# Shifumi by SpliiT Dev

## Description
Ceci est une implémentation simple du jeu de pierre-papier-ciseaux en utilisant HTML, CSS et JavaScript. Les joueurs peuvent affronter l'ordinateur dans une série de duels et voir qui remporte le plus de manches.

## Comment jouer
1. Ouvrez le fichier `index.html` dans votre navigateur web.
2. Choisissez votre coup en cliquant sur la pierre, le papier ou les ciseaux.
3. L'ordinateur choisira également un coup.
4. Les résultats du tour seront affichés, et la manche sera attribuée au vainqueur.

## Structure du Projet
- `index.html` : Fichier principal HTML.
- `style.css` : Feuille de style pour la mise en page et le style du jeu.
- `shifumi.js` : Code JavaScript pour la logique du jeu.
- `script.js` : Code JavaScript pour le reste de la page.
